# VinDex

VinDex is a full-stack vehicle management and VIN lookup system.

## Tech Stack

-   **Backend:** ASP.NET Core, PostgreSQL, Repository Pattern, Dependency Injection, API Caching
-   **Frontend:** Angular (TypeScript), Nginx
-   **Infrastructure:** Docker

------------------------------------------------------------------------

## Features

-   VIN decoding with NHTSA API (engine, manufacturing, vehicle info)
-   Safety recall lookup and market value estimation
-   In-memory caching for fast lookups
-   Secure authentication and user management
-   Responsive Angular UI
-   Fully containerized (backend, frontend, database)
